How it works

A resume screening agent, built like a recruiter's checklist.

Every screening result on this site is issued the way a boarding pass is issued — one candidate, one role, one honest verdict. Behind it is a straightforward pipeline: extract text from your resume, hand it to an LLM with a carefully engineered prompt, and turn the response into a score you can act on.

Pipeline

From upload to verdict

01

Resume parsing

Your PDF or DOCX is parsed on the backend (Apache PDFBox / POI) and converted into plain text.

02

Prompt engineering

The resume text and job description are combined into a structured prompt: a recruiter persona, numbered instructions, and a strict JSON output schema — so the model's answer is consistent and easy to parse.

03

LLM screening

The prompt is sent to an LLM (Gemini), which returns a match score, strengths, gaps, and a verdict — grounded only in what's actually written in the resume.

04

Stored & tracked

Every result is saved to MySQL, so your profile always shows your latest score and full screening history.
